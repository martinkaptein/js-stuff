Python CMS is the most simple blog solution in the universe

PythonCMS is a truly simple, yet very flexible static site generator, written in Python 3.

View Github Download

Why Python CMS?

There are many advantages to PythonCMS: It requires only knowledge of very basic HTML (no weird config files) and it runs on any platform.

The philosophy behind PythonCMS

I always wanted to develope a truly simple blogging system that simultaniously offers full control. With PythonCMS, I doubt that things can get any easier than that.

Martin Kaptein
Developer

Furhermore it's by design fully portable as you can simply upload the whole directory to your server. Then you can continue on any platform that supports Python, without installing thousands of dependencies.

You do not need any kind of local webserver for previewing your blog, and theming is very easy as well. Also, there is support for AMP (Accelerated Mobile Pages by Google) and Search Engine Optimization (SEO) is done very easily.

See the README.md for instructions on how to get started.
